# Welcome to CricPR API

CricPR is a powerful REST API designed for managing cricket matches, teams, players, and live scoring. Whether you're building a cricket scoring app, tournament management system, or analytics platform, CricPR provides all the tools you need.

## Key features

<CardGroup cols={2}>
  <Card title="Multiple Match Formats" icon="calendar">
    Support for Test, ODI, T20, and custom limited-overs formats with ICC-compliant rules
  </Card>
  <Card title="Real-time Scoring" icon="bolt">
    Ball-by-ball tracking with live updates via WebSocket connections
  </Card>
  <Card title="Comprehensive Stats" icon="chart-bar">
    Detailed player and team statistics with ICC-style scorecards
  </Card>
  <Card title="Authentication & Security" icon="lock">
    JWT-based authentication with role-based access control
  </Card>

## API base URL

All API endpoints are prefixed with `/v1`:

```
https://api.cricpr.com/v1
```
