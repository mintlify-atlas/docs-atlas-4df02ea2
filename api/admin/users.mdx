---
title: "Admin - Manage Users"
api: "GET /v1/admin/players"
description: "Admin endpoint to list and filter users/players"
---

Retrieve a paginated and filterable list of all players in the system. This endpoint is restricted to admin users only.

## Authentication

This endpoint requires admin authentication via Bearer token.

```bash
Authorization: Bearer YOUR_ADMIN_ACCESS_TOKEN
```

## Query Parameters

<ParamField query="page" type="integer" default="1">
  Page number for pagination
</ParamField>

<ParamField query="limit" type="integer" default="20">
  Number of players per page
</ParamField>

<ParamField query="name" type="string">
  Filter by player name (partial match)
</ParamField>

<ParamField query="city" type="string">
  Filter by city
</ParamField>

<ParamField query="battingStyle" type="string">
  Filter by batting style. Options:
  - `Left-hand bat`
  - `Right-hand bat`
  - `none`
</ParamField>

<ParamField query="bowlingStyle" type="string">
  Filter by bowling style. Options:
  - `Right-arm fast`
  - `Right-arm medium`
  - `Left-arm fast`
  - `Left-arm medium`
  - `Slow left-arm orthodox`
  - `Slow left-arm chinaman`
  - `Right-arm Off Break`
  - `Right-arm Leg Break`
  - `none`
</ParamField>

<ParamField query="role" type="string">
  Filter by playing role. Options:
  - `Top-order batter`
  - `Middle-order batter`
  - `Lower-order batter`
  - `Opening batter`
  - `Wicket-keeper batter`
  - `Wicket-keeper`
  - `Bowler`
  - `All-rounder`
  - `none`
</ParamField>
