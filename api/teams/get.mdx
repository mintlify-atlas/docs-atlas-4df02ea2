---
title: 'Get Team'
api: 'GET /v1/teams/:teamId'
---

Retrieve details of a specific team.

## Path Parameters

<ParamField path="teamId" type="string" required>
  The ID of the team to retrieve
</ParamField>

## Response

<ResponseField name="id" type="string">
  Team ID
</ResponseField>

<ResponseField name="name" type="string">
  Team name
</ResponseField>

<ResponseField name="logo" type="string">
  Team logo URL
</ResponseField>

<ResponseField name="location" type="string">
  Team location
</ResponseField>

<ResponseField name="captain" type="object">
  Captain player details
</ResponseField>

<ResponseField name="players" type="array">
  Array of player objects
</ResponseField>

<ResponseField name="createdBy" type="string">
  Player ID of the creator
</ResponseField>

<RequestExample>
```bash cURL
curl -X GET https://api.cricpr.com/v1/teams/688a38f0fea3456a96402a30
```
