---
title: 'Get Match Team Stats'
api: 'GET /v1/stats/match/{matchId}/players'
---

Retrieve statistics for all players in a match, organized by team.

<Note>This endpoint requires authentication with `getStats` permission.</Note>

## Path Parameters

<ParamField path="matchId" type="string" required>
  MongoDB ObjectId of the match
</ParamField>

## Response

<ResponseField name="matchId" type="string">
  Match ID
</ResponseField>

<ResponseField name="team1" type="object">
  Team 1 statistics
  <ResponseField name="teamId" type="string">
    Team ID
  </ResponseField>
  <ResponseField name="teamName" type="string">
    Team name
  </ResponseField>
  <ResponseField name="totalRuns" type="integer">
    Total runs scored by team
  </ResponseField>
  <ResponseField name="totalWickets" type="integer">
    Total wickets lost
  </ResponseField>
  <ResponseField name="overs" type="number">
    Total overs faced
  </ResponseField>
  <ResponseField name="players" type="array">
    Array of player statistics
  </ResponseField>
</ResponseField>

<ResponseField name="team2" type="object">
  Team 2 statistics (same structure as team1)
</ResponseField>

## Example Request

```bash
curl -X GET "https://api.cricpr.com/v1/stats/match/507f1f77bcf86cd799439011/players" \
  -H "Authorization: Bearer YOUR_API_TOKEN" \
  -H "Content-Type: application/json"
```
