---
title: "Get Match Scorecard"
api: "GET /v1/matches/{matchId}/scorecard"
---

Retrieve comprehensive match scorecard with ICC-style batting and bowling statistics for each team.

## Path Parameters

<ParamField path="matchId" type="string" required>
  Match ID (MongoDB ObjectId)
</ParamField>

## Query Parameters

<ParamField query="forceRefresh" type="boolean" default={false}>
  Force refresh the cached scorecard data
</ParamField>

<ParamField query="skipCache" type="boolean" default={false}>
  Skip cache and fetch fresh data
</ParamField>

<ParamField query="skipStatsUpdate" type="boolean" default={false}>
  Skip updating player statistics
</ParamField>

## Response

<ResponseField name="match" type="object">
  Match information
  <ResponseField name="id" type="string">
    Match ID
  </ResponseField>
  <ResponseField name="title" type="string">
    Match title
  </ResponseField>
  <ResponseField name="status" type="string">
    Match status: `pending`, `inProgress`, `completed`, `cancelled`, `postponed`
  </ResponseField>
  <ResponseField name="startdatetime" type="string">
    Match start date and time
  </ResponseField>
  <ResponseField name="endDateTime" type="string">
    Match end date and time
  </ResponseField>
</ResponseField>

<ResponseField name="teams" type="array">
  Array of team scorecards (one per innings)
  <ResponseField name="teamName" type="string">
    Team name
  </ResponseField>
  <ResponseField name="teamLogo" type="string">
    Team logo URL
  </ResponseField>
  <ResponseField name="runs" type="integer">
    Total runs scored
  </ResponseField>
  <ResponseField name="wickets" type="integer">
    Total wickets lost
  </ResponseField>
  <ResponseField name="overs" type="string">
    Overs bowled (e.g., "20.0" or "19.3")
  </ResponseField>
  <ResponseField name="totalOvers" type="integer">
    Total overs allocated
  </ResponseField>
  <ResponseField name="battingStats" type="array">
    ICC-style batting statistics
    <ResponseField name="playerName" type="string">
      Player name (uses `shortName` for guest players)
    </ResponseField>
    <ResponseField name="dismissal" type="string">
      Dismissal description (e.g., "caught Smith b Johnson", "not out")
    </ResponseField>
    <ResponseField name="runs" type="integer">
      Runs scored
    </ResponseField>
    <ResponseField name="balls" type="integer">
      Balls faced
    </ResponseField>
    <ResponseField name="fours" type="integer">
      Number of fours
    </ResponseField>
    <ResponseField name="sixes" type="integer">
      Number of sixes
    </ResponseField>
    <ResponseField name="strikeRate" type="number">
      Strike rate (runs per 100 balls)
    </ResponseField>
    <ResponseField name="isNotOut" type="boolean">
      Whether the player is not out
    </ResponseField>
  </ResponseField>
  <ResponseField name="bowlingStats" type="array">
    ICC-style bowling statistics
    <ResponseField name="playerName" type="string">
      Player name (uses `shortName` for guest players)
    </ResponseField>
    <ResponseField name="overs" type="number">
      Overs bowled
    </ResponseField>
    <ResponseField name="maidens" type="integer">
      Maiden overs
    </ResponseField>
    <ResponseField name="runs" type="integer">
      Runs conceded
    </ResponseField>
    <ResponseField name="wickets" type="integer">
      Wickets taken
    </ResponseField>
    <ResponseField name="noBalls" type="integer">
      Number of no balls
    </ResponseField>
    <ResponseField name="wides" type="integer">
      Number of wides
    </ResponseField>
    <ResponseField name="dots" type="integer">
      Number of dot balls
    </ResponseField>
    <ResponseField name="economy" type="number">
      Economy rate (runs per over)
    </ResponseField>
    <ResponseField name="average" type="number">
      Bowling average (runs per wicket)
    </ResponseField>
  </ResponseField>
</ResponseField>

## Guest Players in Scorecard

Both regular and guest players appear in batting and bowling statistics:
- **Guest players**: Display name from `shortName`, no profile picture
- **Regular players**: Display name from `shortName` or `userId.name`

## Example Request

```bash
curl -X GET https://api.cricpr.com/v1/matches/507f1f77bcf86cd799439011/scorecard
```

```bash
# Force refresh cached data
curl -X GET "https://api.cricpr.com/v1/matches/507f1f77bcf86cd799439011/scorecard?forceRefresh=true"
```

## Error Codes

  **Bad Request** - Invalid match ID format
</ResponseField>
