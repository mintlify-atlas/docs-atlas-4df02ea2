This guide explains how to retrieve and interpret cricket statistics from the CricPR API.

## Overview

The API provides multiple levels of statistics:

1. **Match statistics** - Overall match data and scorecard
2. **Player statistics** - Batting, bowling, and fielding stats
3. **Innings statistics** - Per-innings breakdowns
4. **Live statistics** - Real-time updates during matches

## Match statistics

Get comprehensive match statistics including scorecard and player performance:

```javascript
GET /v1/matches/:matchId/stats
```

**Response:**

```json
{
  "match": {

    "title": "Summer League Final",
    "status": "completed",
```

## Player statistics

### Batting statistics

Get detailed batting statistics for a player in a match:

```json
{
  "playerId": "64a1234567890abcdef12350",
  "matchId": "64a1234567890abcdef12347",
  "battingStats": {
    "runScored": 65,
    "ballsFaced": 42,
    "fours": 7,
    "sixes": 2,
    "strikeRate": 154.76,
    "dotBalls": 15,
    "singles": 20,
    "twos": 8,
    "threes": 2
  }
}
```

From `src/services/stats.service.js:128-200`, the API uses aggregation for optimal performance:

```javascript
const calculateAllPlayerStats = async (innings, overs) => {
  // Get all balls from all overs
  const allBalls = overs.reduce((acc, over) => [...acc, ...over.balls], []);

  // Group balls by player (both batting and bowling)
  const playerBalls = {};
  allBalls.forEach((ball) => {
    // Group by batsman
    const batsmanId = ball.batsmanId.toString();
    if (!playerBalls[batsmanId]) {
      playerBalls[batsmanId] = { batting: [], bowling: [] };
    }
    playerBalls[batsmanId].batting.push(ball);

    // Group by bowler
    const bowlerId = ball.bowlerId.toString();
    if (!playerBalls[bowlerId]) {
      playerBalls[bowlerId] = { batting: [], bowling: [] };
    }
    playerBalls[bowlerId].bowling.push(ball);
  });
};
```

## Stats calculation

The API recalculates statistics automatically when:

1. A ball is recorded
2. A ball is updated or undone
3. An innings ends
4. A match ends

From `src/services/stats.service.js:96-120`:

```javascript
const updateInningsStats = async (inningsId) => {
  const inningsData = await Innings.findById(inningsId)
    .populate('battingTeamInstanceId bowlingTeamInstanceId');

  // Get all overs for this innings
  const overs = await Overs.find({ inningsId }).populate('balls');

  // Calculate all player stats (batting and bowling)
  const results = await calculateAllPlayerStats(inningsData, overs);

  return {
    inningsId: inningsData._id,
    results,
  };
};
```

<Note>
Stats updates run asynchronously using background workers to avoid blocking API responses during live scoring.
</Note>

## Caching strategy

Statistics are cached with different TTL based on match status:

<Steps>

### Completed matches

Cached for 24 hours (86400 seconds):

```javascript
const ttl = match?.status === 'completed' ? 86400 : 300;
```

### Live matches

Cached for 5 minutes (300 seconds) for real-time accuracy

### Force refresh

Bypass cache when needed:

```javascript
GET /v1/matches/:matchId/stats?forceRefresh=true
```

</Steps>

## Player milestones

The API automatically tracks and notifies players of milestones:

- **Batting**: 30 runs, half-century (50), century (100)
- **Bowling**: 3 wickets, fifer (5 wickets)
- **Player of the Match**: Calculated based on performance score

From `src/services/match.service.js:382-522`:

```javascript
const sendPlayerMilestoneNotifications = async (matchId) => {
  // Get all player match stats
  const playerMatchStats = await PlayerMatchStats.find({ matchId })
    .populate('playerId')
    .populate('battingStatsId')
    .populate('bowlingStatsId');

  // Check for milestones
  for (const playerStat of playerMatchStats) {
    const battingStats = playerStat.battingStatsId;
    const bowlingStats = playerStat.bowlingStatsId;
    const milestones = [];

    // Batting milestones
    if (battingStats) {
      const runs = battingStats.runScored || 0;
      if (runs >= 100) {
        milestones.push('Scored Century');
      } else if (runs >= 50) {
        milestones.push('Scored Half Century');
      }
    }

    // Bowling milestones
    if (bowlingStats) {
      const wickets = bowlingStats.wickets || 0;
      if (wickets >= 5) {
        milestones.push('Took Fifer');
      }
    }
  }
};
```
