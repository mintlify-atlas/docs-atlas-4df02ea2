This guide explains how to connect to the CricPR API's WebSocket service for real-time match updates.

## Overview

The CricPR API provides real-time updates through Socket.io, enabling:

1. Live score updates as balls are bowled
2. Instant wicket notifications
3. Match status changes
4. Chat and messaging between users

<Note>
The WebSocket service runs alongside the REST API and shares the same authentication.
</Note>

## Connection

Connect to the Socket.io server:

```javascript
import io from 'socket.io-client';

const socket = io('https://api.cricpr.com', {
  transports: ['websocket'],
  auth: {
    token: 'your-jwt-token'
  }
});

socket.on('connect', () => {
  console.log('Connected to CricPR WebSocket');
});

socket.on('disconnect', () => {
  console.log('Disconnected from CricPR WebSocket');
});
```

## User presence

Track active users in the application:

<Steps>

### Announce presence

Notify server when user becomes active:

```javascript
socket.emit('active', userId);
```

### Listen for active users

Receive list of currently active users:

```javascript
socket.on('activeUsers', (users) => {
  console.log('Active users:', users);
  // users: Array of user objects with profile info
});
```

### Request active users

Manually request the current list:

```javascript
socket.emit('activeUsers');
```

</Steps>

From `src/services/socket.service.js:22-74`:

```javascript
socket.on('active', async (userId) => {
  userIds.push(`${socket.id}split${userId}`);
  const userDocs = await User.find({ _id: { $in: userIds.map((itm) => itm.split('split')[1]) } })
    .select('profilePicture name email')
    .exec();
  
  const finalData = await Promise.all(
    userDocs.map(async (itm) => {
      return {
        profilePicture: itm.profilePicture,
        name: itm.name,
        email: itm.email,
        id: itm._id,
      };
    })
  );

  io.emit('activeUsers', finalData);
});
```

```javascript
socket.on('groupMessage', async ({ chatId, message, senderId, content_type: contentType }) => {
  const chat = await Chat.findById(chatId);
  if (chat.active) {
    const newMessage = new Message({
      chat_id: chatId,
      sender_id: senderId,
      content: message,
      content_type: contentType,

    await newMessage.save();
    
    chat.lastMessage = newMessage._id;
    await chat.save();

    io.to(chatId).emit('groupMessage', {

      time: newMessage.createdAt,

      content_type: newMessage.content_type,
```

```javascript
socket.on('editMessage', async ({ messageId, newContent }) => {
  try {
    const message = await Message.findById(messageId);
    if (!message) {
      return;
    }
    const chat = await Chat.findById(message.chat_id);
    if (chat.active) {
      message.content = newContent;
      await message.save();

      io.to(message.chat_id).emit('editMessage', {
        message: message.content,
        time: message.createdAt,
        sender_id: message.sender_id,
        read: message.read,
        content_type: message.content_type,
      });
    }
  } catch (error) {
    throw new Error();
  }
});
```

## Complete example

Here's a complete React Native example:

```javascript
import React, { useEffect, useState } from 'react';
import io from 'socket.io-client';
import AsyncStorage from '@react-native-async-storage/async-storage';

export default function LiveMatch({ matchId }) {
  const [socket, setSocket] = useState(null);
  const [score, setScore] = useState({ runs: 0, wickets: 0, overs: 0 });
  const [isConnected, setIsConnected] = useState(false);

  useEffect(() => {
    const initSocket = async () => {
      const token = await AsyncStorage.getItem('auth_token');
      const userId = await AsyncStorage.getItem('user_id');

      const newSocket = io('https://api.cricpr.com', {
        auth: { token },
        transports: ['websocket']
      });

      newSocket.on('connect', () => {
        setIsConnected(true);
        newSocket.emit('join', { userId, matchId });
      });

      newSocket.on('disconnect', () => {
        setIsConnected(false);
      });

      newSocket.on('scoreUpdate', (data) => {
        setScore({
          runs: data.runs,
          wickets: data.wickets,
          overs: data.overs
        });
      });

      newSocket.on('wicket', (data) => {
        alert(`Wicket! ${data.batsman} is out`);
      });

      setSocket(newSocket);
    };

    initSocket();

    return () => {
      if (socket) {
        socket.disconnect();
      }
    };
  }, [matchId]);

  return (
    <View>
      <Text>Status: {isConnected ? 'Live' : 'Offline'}</Text>
      <Text>Score: {score.runs}/{score.wickets} ({score.overs} overs)</Text>
    </View>
  );
}
```
